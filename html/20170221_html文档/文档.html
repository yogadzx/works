doctype

    DOCTYPE:用于告诉浏览器处理的是html文档，告诉html版本
    由于html5之前基于sgml（标准通用标记语言），所以需要声明引用DTD（文档类型定义）
    html5 不基于sgml，所以不需要dtd。
    html5: <!DOCTYPE html>
    html4: < !DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Frameset//EN""http://www.w3.org/TR/html4/frameset.dtd" >
