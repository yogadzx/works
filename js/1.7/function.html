使用函数声明语句可提前

使用函数表达式，不可在表达式之前调用函数
var fun = function (){}

函数调用

表达式普通调用，在ecam3和非严格的ecma5下，调用函数的上下文是全局对象，严格模式为undefined
当方法不需要返回值时，最好返回this，方便方法链。

this

this不具有继承性，嵌套函数不会保留this，当嵌套函数被调用时，跟普通函数调用一样。
普通调用this不是全局就是undefined，方法调用时，this是调用他的对象。

构造函数调用

如果构造函数没有形参，那么调用构造函数时可省略（）；
如果构造函数有 return 一个对象，那么构造函数返回这个对象。
var o = {name : "what"} function structor(){ this.name ="dzx"; return o }
